搭建博客

发表于 2021-09-26 分类于 -搭建博客 Valine：

hexo搭建博客

本地搭建

环境：windows10

安装Node.js

进入官网Node.js,下载安装包,默认安装即可。
打开官网git
按照默认安装即可
git安装完成后，需要进行配置
输入如下，其中” “中的your name 和your email为你的Git Hub用户名(非昵称)与邮箱（这样也方便以后在Github的使用）

git config --global user.name "your name"
git config --global user.email "your email"

并可通过以下命令查询用户名与邮箱

git config user.name
git config user.email

安装hexo

在本地找一个地方做为我们Hexo的数据数据存放，例如我的存放目录为D:\yun\blog，我们在这个目录下右键打开Git Bash Here
先查看我们的Node 和 npm有没有安装好

node -v
npm -v


安装个cnpm使下载指向国内源以提高速度。
这里我们选择使用淘宝镜像下载 cnpm，在上面终端继续输入。

npm install -g cnpm --registry=https://registry.npm.taobao.org

下载完后查看cnpm版本，查询成功则证明安装完成。

cnpm -v

正式的开始下载Hexo，使用cnpm下载hexo，并验证是否成功

cnpm install -g hexo-cli
hexo -v

安装组件

cnpm install hexo --save

我们需要进行Hexo的初始化（注意我们的路径）。我们可以使用pwd查看该目录是否为我们希望的地址。

hexo init

主题的启用

在主题下载网址https://hexo.io/themes/下载喜欢的类型
下载到blog下的themes下
在blog下的_config.yml配置

theme： next

配置主题内的设置，我设置的是next主题，参考主题设定配置
设置语言 

language: zh-CN

设置菜单栏

title: MyBlog
subtitle: ''
description: ''
keywords:
author: xnh
language: zh-CN
timezone:

根据自己喜欢设置头像和主题样式
在next下

url: /images/avatar.gif    设置头像
scheme：Pisces      设置主题

可以选择添加社交栏，在next下的_config.yml设置social

social:
GitHub: https://github.com/xnhzs || github
Weibo: https://weibo.com/u/7520731506/home || weibo

之后在Git Bash Here下

hexo n name

创建新文章在文章下设置tags和categories选择分类和标签,这样一篇文章的基本框架便有了
